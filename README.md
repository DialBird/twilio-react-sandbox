# Twilio VideoのSandbox

- [こちら](https://github.com/twilio/twilio-video-app-react)をVanila JSにかきなおしつつ、構成を理解するためのもの
- Twilioにプロジェクトを作成して、`.env.example`に必要な値を入れて`.env`に改名して`npm start`すれば動く
- 以下の機能を実現できる
  - 自分の名前とルーム名を指定して、会話通信
  - 画面のオンオフ
  - 音声のオンオフ
  - 共有画面のオンオフ
